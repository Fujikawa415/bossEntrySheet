### タスク管理

-   追加
-   消化
-   確認
-   急かす機能

```
タスクは大きな単位で書いて、紐づく小タスクを説明欄に列挙してください
工数は直感より余分を持たせつつも、余分を正当化できるよう小タスクを細かく列挙してください

8時間で4タスクなら1タスク2時間ってことになる

タスク追加
ユーザーに紐づくタスクが1つなら開始
ユーザーに紐づくタスクが2つ以上ならセレクトさせる
```

必要そうな項目

-   タスク ID
-   タスク名
-   タスクの説明　自由記入
-   予定工数
-   実績工数
-   ユーザー ID
-   開始日時
-   終了日時
